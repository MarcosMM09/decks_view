# decks_view

El siguiente repositorio contiene el proyecto de "Reto Android", el cual cumple con las indicaciones mencionadas en el documento proporcionado.

INSTRUCCIONES:
- La aplicacion cuenta con componentes realizados en jetpack compose (Cards, Text, Topbar, LazyColumn, DropdownMenu, etc).
- La aplicacion como primer paso comenzará a consumir el servicio de tarjetas proporcionado para la practica "https://db.ygoprodeck.com/api/v7/cardsets.php"
- Para el consumo del servicio se utilizó Ktor, Arquitectura MVVM y JetpackCompose.
- Una vez que el servicio responde la app comienza a pintar las tarjetas, las cuales tienen  una opcion de agregar a afavoritos.
- si se le da clic a la opcion de favoritos y no se ha agregado nada simplemente mostrara la coleccion en blanco.
- si se agrega alguna carta a favoritos esta se guarda con Room y crea una lista nueva para guardar y moestrar en el apartado de favoritos.
- El consumo se hace siempre cuando se inicia la app, si la app no tiene internet y ya se consumio una vez el servicio este persiste para poder iniciar las funciones.
